### メモリ割り当て

このmodpackはメモリ割り当てが最小でも4GB必要です。8GB割当すれば、よりよく遊べると思います。  
メモリ割り当てのやり方は下の手順を参考にしてください。  

    <Card title="CurseForge Launcher">
        <Steps>
            1. CurseForge Launcherの設定画面を開き、左のメニューから「Minecraft」を選択する。
            2. 「割り当て済みメモリ」と書いてある下のつまみを調整して自分の好きなようにメモリを割り当てする。
        </Steps>
    </Card>
    <Card title="Prism Launcher">
        <steps>
            1. Prism Launcher上部の「設定」を押し、左のメニューから「Java」を選択する。
            2. 「General」のタブを押し、「最大メモリ割り当て」で好きなようにメモリを割り当てする。(4192 MiB = 4GB)
        </steps>
    </Card>
    <Card title="その他のランチャー">
        その他のランチャー(GDLauncherなど)を使用している場合は、Google検索などを用いて設定してください。
    </Card>

### ランチャー

一般的なランチャーは[CurseForge](https://www.curseforge.com/download/app)や[Prism](https://prismlauncher.org)です。パックをインストールするには(手動でもできますが)これらのランチャーが必要です。  
Microsoftアカウントにログインし、「takoyaki-s-modpack」と探し、一番上に出てきたmodpackをダウンロードします。
